
# AKS Microservices Deployment

## Project Overview

This project demonstrates deployment of microservices architecture on Azure Kubernetes Service (AKS) using Docker, Kubernetes, Jenkins CI/CD pipelines, and Azure Container Registry.

---

## Technologies Used

- Azure AKS
- Docker
- Kubernetes
- Jenkins
- Azure Container Registry
- Java
- YAML

---

## Features

- Microservices deployment
- CI/CD pipeline automation
- Docker image build and deployment
- Kubernetes ingress configuration
- Load-balanced frontend service
- Backend internal communication

---

## Architecture Flow

Developer → GitHub → Jenkins → Docker → ACR → AKS Cluster

---

## Future Enhancements

- Helm charts
- Monitoring with Prometheus & Grafana
- SSL/TLS integration
- GitHub Actions
- Terraform automation
